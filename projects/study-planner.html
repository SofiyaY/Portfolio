A clean, focused desktop Study Planner built with JavaFX and Gradle. Quickly add tasks with a title, due date, and a time estimate, then track completion in a responsive table view. The app includes a simple Home screen, an accessible inline form, keyboard-friendly interactions, and a polished light theme.

View Repository Back to Projects

Highlights

  • Modern JavaFX UI with a Home screen and Tasks view
  • Inline add form with validation (Title, Due Date, Estimate minutes)
  • TableView with readable headers, date formatting, and checkbox completion
  • Clean light theme: readable text, subtle shadows, polished buttons
  • Gradle build; Java 17; FXML + CSS separation

What I built

  • Model: Task with JavaFX properties (title, dueDate, estimateMinutes, completed)
  • Controller: Binds columns, formats dates, aligns estimate values, and handles add/delete
  • Navigation: HomeController switches to Tasks while preserving styling
  • Styling: application.css for the overall palette, hover states, and readable prompts
