How to Run (Locally)

You need Node.js 18+ and Java 17+ installed.

Backend

# From the project root
cd backend
./mvnw spring-boot:run
# API runs on http://localhost:8080

Frontend

# In a new terminal
cd frontend
npm install
npm run dev
# Opens on http://localhost:5173

The frontend includes sample data in localStorage, so you can explore without starting the backend.
